## 描述 Description
本服务器整合包项目，旨在制作一个：
- 将武器工具的锻造系统，全部基于**匠魂**形式重做；
- 魔法附魔系统全部基于原版附魔重做；
- 自动化系统主要依赖**机械动力**模组；
- 战斗系统基于**史诗战斗**模组；
- 致力于：生存、建筑休闲玩家体验细节优化；<br>

**主线游戏任务风格为战斗、魔法与机械动力**的1.20.1版本整合包。<br>
